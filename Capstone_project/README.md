# Foodie — Food Delivery App (Capstone)

## Project Title & Theme
**Foodie** — a static, responsive Food Delivery landing/menu page built using HTML5 & CSS3.

## Description
This is a single-page static site demonstrating layout, semantic HTML, responsive design (Flexbox + Grid), forms, cards, and visual polish suited for a food delivery app landing page.

## Sections implemented
- Header / Navigation (logo, links, CTAs)
- Hero / Banner with CTA
- Featured Menu (card grid)
- Filters (visual aside)
- Restaurants list
- Testimonials
- Contact / Order form
- Footer

## Breakpoints & techniques
- Desktop: default ≥1024px (3-column grid for menu)
- Tablet: ≤1024px (2-column grid)
- Mobile: ≤768px (single-column layout) and ≤480px (small phone tweaks)
- Techniques used: CSS Grid (menu), Flexbox (header/footer), CSS variables, transitions, and a keyframe animation.
